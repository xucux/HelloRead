# HelloRead

IntelliJ IDEA 小说阅读插件

## 功能

- 📚 书架管理
- 📖 智能阅读器  
- 🔍 章节自动解析
- ⚙️ 个性化设置

## 快捷键

- `Ctrl+Alt+B` - 打开书架
- `Ctrl+Alt+A` - 添加书籍
- `Ctrl+Alt+R` - 开始阅读
- `Ctrl+Alt+1/2` - 上一章/下一章
- `Ctrl+Alt+4/5` - 阅读界面上下滚动

## 安装

从 IntelliJ IDEA 插件市场搜索 "HelloRead" 安装

## 支持

- UTF-8 编码的 .txt 文件
- 中文章节格式（第X章、第X节等）
